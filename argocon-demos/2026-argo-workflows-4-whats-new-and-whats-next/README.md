## Abstract

A whirlwind tour of the main features of Argo Workflows from the last year, covering UI markdown, performance improvements, CustomResourceDefinitions, multi-controller synchronization, and artifact plugins. The talk concludes with a roadmap preview addressing workflow tracing, unified expression engine rework for improved debuggability, and a pluggable DAG engine.

## Abstract

Argo Workflows 4.0 introduces artifact plugins enabling integration of any storage backend without modifying Argo's core. This talk covers creating plugins as containerized gRPC servers running as sidecars and init containers, alongside discussion of pod management complexities and how Argo Workflows orchestrates pods and artifacts.
